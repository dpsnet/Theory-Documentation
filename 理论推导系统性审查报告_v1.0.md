#### 问题P-001：$c_{eff}$公式缺乏推导

**文档位置**：第59-61行

**当前表述**：
```markdown
光速的有效测量值依赖于谱维：

$$c_{\text{eff}}(d_s) = c \cdot f(d_s)$$

其中$f(d_s) = \sqrt{d_s/4}$是谱维依赖的修正因子。
```

**问题分析**：
1. **无推导过程**：直接给出公式，没有从第一性原理推导
2. **来源不明**：$\sqrt{d_s/4}$的形式是基于什么物理或数学原理？
3. **量纲分析**：$d_s$是无量纲的谱维，如何自然导出速度修正？

**可能来源**（推测）：
- 类比高维空间中的体积元比例？
- 扩散方程的特征速度？
- 纯粹的唯象假设？

**建议修订方案**：

**方案A：严格推导**（首选）
从Dirac算子的谱性质严格推导有效传播速度：
```
从热核迹 Tr(e^{-tD²}) ~ t^{-d_s/2}
    ↓
扩散系数与谱维的关系 D ~ d_s/4
    ↓
有效速度 c_eff ~ c·√(d_s/4)
```

**方案B：明确标注为假设**（如无法严格推导）
```markdown
**假设**（唯象）：光速与谱维的关系为
$$c_{\text{eff}}(d_s) = c \sqrt{\frac{d_s}{4}}$$

**物理动机**：[解释为什么是这个形式]
**验证方式**：[如何通过实验验证/证伪]
**限制**：[此假设的适用范围]
```

**方案C：引入额外假设**
如果现有数学框架无法推导，需要：
1. 明确引入额外物理假设
2. 讨论假设的合理性
3. 探索替代假设的可能性

#### 问题P-002：洛伦兹变换修正的形式一致性

**文档位置**：第99-112行

**当前表述**：
```markdown
时间变换：
$$t' = \gamma \left( t - \frac{vx}{c^2} \right) \cdot f(d_s)$$

空间变换：
$$x' = \gamma (x - vt) \cdot g(d_s)$$

其中：
- $f(d_s) = \sqrt{d_s/4}$ 是时间修正因子
- $g(d_s) = \sqrt{d_s/4}$ 是空间修正因子
```

**问题分析**：
1. 时间和空间使用相同的修正因子是否自洽？
2. 修正后的变换是否保持时空线元不变？
3. 是否满足相对性原理（不同惯性系等价）？

**需要验证**：
- 修正后的洛伦兹变换是否构成群？
- 是否保持因果结构？
- 与实验观测（如时间膨胀、长度收缩）的定量对比
